# Backrow Centre

## Project
Demo portfolio website for Nathan McCrory's **BackRowCentre Media** brand.
Young creative based in Norfolk, UK — video production, photography, creative content.

## Stack
React 19 | TypeScript | Vite | Tailwind CSS 4 | Framer Motion (motion) | Lucide Icons

## Architecture
Single-page app with component-based architecture:
- `src/components/` — Nav, Hero, WorksGrid, Services, About, Testimonials, Contact, Footer, CustomCursor
- `src/data/` — projects.ts, services.ts (content data)
- Dark cinematic aesthetic (ink/paper color tokens) with grain texture overlay

## Deploy
GitHub → Vercel auto-deploy

## Scripts
- `npm run dev` — dev server on port 3000
- `npm run build` — production build
- `npm run lint` — TypeScript check
