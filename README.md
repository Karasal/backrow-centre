# BackRowCentre Media

Demo portfolio website for BackRowCentre Media — visual storytelling through film, photography, and creative content. Based in Norfolk, UK.

## Stack

- React 19 + TypeScript
- Vite 6
- Tailwind CSS 4
- Framer Motion
- Lucide Icons

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
